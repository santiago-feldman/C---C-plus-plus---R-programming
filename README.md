# C, C++ and R programming | ITBA

This repository contains the practical assignments done for the following subjects at Instituto Tecnológico de Buenos Aires:

- 22.07 Programming I
- 22.08 Data Structures and Algorithms
- 94.24 Learning Methodology

## Data Structures and Algorithms (C++ projects)

- **Level 1 | Warm Up** - 3D Solar System simulation both in C and C++ using the raylib library
- **Level 2 | EDAPark** - Skating robot's brushless DC motor control over IoT protocols (MQTT/TCP-IP) inside a simulator built in Unity
- **Level 3 | Lequel** - Language identification using cosine similarity over character trigram profiles
- **Level 4 | EDA-Man** - Pac-Man implementation with autonomous ghosts and PID motion control
- **Level 5 | DNATool** - Needleman-Wunsch algorithm for SARS-CoV-2 genome sequence alignment
- **Level 6 | EDACup** - Autonomous multi-robot soccer, from a goal-scoring technical challenge to a full team competing in an event streamed live to the university community and the general public
- **Level 7 | EDAoogle** - Web search engine over a Wikipedia corpus, built on an inverted index and served over HTTP with an HTML/CSS front-end
